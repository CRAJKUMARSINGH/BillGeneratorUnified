HOW TO USE
==========

1. Place Excel files in 'input/' folder
2. Run BATCH_RUN.bat (or python batch_run_demo.py)
3. Find outputs in 'output/' folder with timestamped names
4. Each Excel file gets its own folder with HTML and PDF subfolders
