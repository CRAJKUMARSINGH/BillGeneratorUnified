## Summary Report

The summary Excel file contains:
- File name
- Iteration number
- Timestamp
- Status (success/error)
- File counts (HTML, PDF, DOC, Excel)
- Output folder path
- Error messages (if any)

## Statistics

The agent will show:
- Total runs
- Successful runs
- Failed runs
- Success rate
- Total files generated by type
